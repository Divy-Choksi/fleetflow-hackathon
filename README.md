# FleetFlow – Modular Fleet & Logistics Management System

FleetFlow is a centralized digital platform designed to replace manual fleet logbooks. It enables organizations to manage vehicles, assign trips, monitor operational status, and ensure safe load handling through capacity validation.

Key Features:
- Role-based login (Manager / Dispatcher)
- Vehicle registry (CRUD)
- Trip dispatch with capacity validation
- Real-time vehicle status (Available / On Trip / In Shop)
- Maintenance tracking
- Operational dashboard

Tech Stack:
Frontend: React / HTML-CSS-JS
Backend: PHP
Database: PHPMyAdmin
